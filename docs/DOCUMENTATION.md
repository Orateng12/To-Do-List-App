## 🎯 Key Concepts

### 1. Identity-First Architecture

**Traditional**: Goals → Tasks → Actions  
**Clear**: Identity → Systems → Habits → Actions

Clear focuses on **who you become**, not just what you achieve.

**Read More**: [ADR-001](./architecture/adr/001-identity-first.md)

---

### 2. User Archetypes

8 distinct patterns that guide your Clear experience:

- 🌱 **Rebuilder** - Starting fresh
- 🧭 **Explorer** - Seeking direction
- 🏔️ **Striver** - Ambitious achiever
- 💪 **Physical Transformer** - Health-focused
- 📈 **Career Climber** - Professional growth
- ⚡ **Life Optimizer** - Maximum efficiency
- 🧘 **Meaning Seeker** - Purpose-driven
- ⚖️ **Balance Builder** - Harmony seeker

**Read More**: [Archetypes Guide](./psychology/archetypes/README.md)

---

### 3. Oblique Metrics

Measure what matters indirectly:

| Metric | What It Measures | Target |
|--------|------------------|--------|
| Identity Reinforcement | Alignment with desired identity | > 70% |
| Consistency | How reliably you show up | > 80% |
| Momentum | Are you accelerating? | Increasing |
| Growth | Overall progress | > 60 |
| Adaptive Load | Appropriate challenge level | 4-6 |

**Read More**: [ADR-002](./architecture/adr/002-oblique-metrics.md)

---

### 4. Daily Execution

**The Hard Thing**: One important task per day  
**Habits**: Small actions that reinforce identity  
**Reflection**: Evening review of alignment

**Read More**: [Daily Usage Guide](./guides/user/daily-usage.md)

## 🔧 Technical Overview

### System Architecture

```
┌─────────────────────────────────────┐
│     Presentation Layer (UI)         │
│     ClearUIRenderer                 │
└──────────────┬──────────────────────┘
               ↓
┌─────────────────────────────────────┐
│     Domain Layer (Business Logic)   │
│     ClearEngine                     │
│     - Archetype System              │
│     - Identity Manager              │
│     - System/Habit Managers         │
│     - Metrics Engine                │
└──────────────┬──────────────────────┘
               ↓
┌─────────────────────────────────────┐
│     Data Layer (Storage)            │
│     In-Memory → LocalStorage →      │
│     IndexedDB (future)              │
└─────────────────────────────────────┘
```

**Read More**: [Architecture Overview](./architecture/overview.md)

---

### Technology Stack

- **Language**: TypeScript (100%)
- **UI**: Vanilla JavaScript (no framework)
- **Storage**: In-memory (LocalStorage/IndexedDB in Phase 2)
- **Build**: Vite
- **Bundle Size**: 43KB (gzipped)
